# myCalculator
A calculator project using HTML, CSS and JavaScript where I designed the layout and appearance of the interface. The goal is to create a calculator that looks like a real one (with buttons and a display screen).  HTML is used to build the structure (buttons, screen, layout). CSS is used to style it, while JavaScript is used to add functionality.
With only HTML and CSS, the calculator will not actually calculate anything.

JavaScript Functionality:

Detect button clicks
Perform calculations
Update the display

💡 What You Learn
HTML layout and structure
CSS styling and grid system
UI (user interface) design basics
JavaScript interactivity
